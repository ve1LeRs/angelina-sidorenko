# Ангелина Сидоренко — сайт-визитка

**Live:** https://angelina-sidorenko.is-a.dev  
**Резерв:** https://ve1lers.github.io/angelina-sidorenko/

Статический одностраничный сайт оперной певицы.

## Локальный запуск

```bash
python -m http.server 8765
```

Открыть: http://localhost:8765

## Обновить сайт в интернете

```powershell
git add .
git commit -m "update"
git push origin main
```

Или: `.\deploy.ps1`

## Файлы

- `index.html` — страница
- `styles.css` — стили
- `script.js` — интерактив
